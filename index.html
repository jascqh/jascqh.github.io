Portfolio

Fake News Detection

Machine Learning, NLP, Full Stack Development

Fake News Detection


The fake news detector detects whether a social media post contains fake news. One of the main features was the stance detection algorithm that was done using various NLP tools and deep learning models such as BERT and roBERTa. The model was integrated into a web application built using React to provide a platform for users to easily use the fake news classifier. The classifier achieved a high f1 accuracy of 80%.

Interactive Dashboard

Data Visualisation, Machine Learning

Interactive Dashboard


We helped our end users to shorten the time taken to review assessment scores of their candidates by introducing interactive dashboards done using Tableau to replace the standard hardcopy reports for each candidate. Users can measure the performance of a candidate through visual clues and can easily compare an individual’s performance to the cohort.


We also used machine learning models such as logistic regression, random forest, naive bayes and neural network to do a predictive modelling for each candidate which we believe will help to speed up the process of shortlisting candidates based on their performances.

We built a python GUI to help the end users to easily train and use the models.

Hotel Reviews

NLP, Machine Learning, Data Exploration

Hotel Reviews


We performed topic modelling using Gensim and sentiment analysis using Afinn on the data of hotel reviews from Kaggle to find out what are the best hotels for different hotel features. (ie. cleanliness, breakfast, service etc) Knowing the best and worst features of a hotel allows the hotel managers to make data driven decisions on what to improve.

We also tried to find out whether our generated topics and sentiment scores can accurately predict the average score for the hotel, therefore providing a good indication of the hotel's overall performance.
